- 브랜치 -
목록 : git branch
생성 : git branch {name}
전환 : git switch {name}
합치기 (오류시 수정후 add->commit) git merge {name}
       git switch {name} => (오류시 수정후 git rebase --continue) git rebase {main_name} => git switch {main_name} => git merge {name}
       (오류시 수정후 add->commit) git merge {name} --squash
       전부다 결과는 같은데 2,3번은 로그가 깔끔하게 나옴
삭제 (머지) git branch -d {name}
    (머지X) git branch -D {name}

- github관련 -
push -f : 강제넣기
pull/clone --allow-unrelated-histories : 강제가져오기
git push -u {url} {main_name}
그냥 가져오기(폴더생김) : git clone {url}
현재 폴더에 적용 : git pull

- 로그초기화 -
